# Learning Templete
This repository contains various learning templates for different programming languages, frameworks and databases. Each template is designed to help you get started quickly with best practices and common patterns.

## Available Templates
- **HTML**: A basic HTML5 template with linked CSS and JavaScript files.
- **CSS**: A starter CSS file with common styles and a reset.
- **Bootstrap**: A template using Bootstrap 5 for responsive design.
- **Tailwind CSS**: A template using Tailwind CSS for utility-first styling.
- **JavaScript**: A simple JavaScript template with ES6 features.
- **TypeScript**: A TypeScript template with configuration for strict type checking.
- **React**: A React application template using Create React App.
- **Vue.js**: A Vue.js application template using Vue CLI.
- **Node.js**: A basic Node.js server setup using Express.
- **MySQL**: A sample database schema and queries for MySQL.
- **PostgreSQL**: A sample database schema and queries for PostgreSQL.
- **SQLite**: A sample database schema and queries for SQLite.
- **MongoDB**: A sample database schema and queries for MongoDB.(NoSQL)
