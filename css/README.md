# CSS Template
This is a basic HTML5 template that includes linked CSS and JavaScript files to help you get started with web development.

## Template Structure
- `index.html`: The main HTML file.
- `styles.css`: A linked CSS file for styling.
- `script.js`: A linked JavaScript file for interactivity.
